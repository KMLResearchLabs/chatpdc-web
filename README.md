# ChatPDC Web

O primeiro ChatBot com distúrbio de personalidade, ele pode assumir diferentes personalidades a depender do que o usuário escolher, no ínício o intuito era ele simplesmente ser burro e engraçado, porém agora ele pode dar diferentes resposta a depender do gosto do usuário.

Outra vantagem é que seu uso é completamente anônimo, qualquer um pode usar sem se preocupar de ter seus dados roubados, e claro é 100% free e forever will be
Ele contem tambem memória por chat, ou seja, a sessão atual ele pode ver o contexto, se reiniciar a página o contexto é excluído
